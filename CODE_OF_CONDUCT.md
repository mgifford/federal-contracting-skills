# Code of Conduct

## Our Standards

This project is a professional tool for the federal acquisition workforce. We expect all participants to:

- Be respectful and constructive
- Focus on improving the tools
- Accept feedback gracefully
- Keep discussions relevant to federal contracting and the skills in this repo

## Scope

This applies to all project spaces: issues, pull requests, and discussions.

## Enforcement

Unacceptable behavior can be reported to the maintainer. Reports will be reviewed and addressed as appropriate.
